
# Bash Script Labs

A collection of practical bash scripting exercises and labs for learning shell scripting fundamentals.

## Overview

This repository contains hands-on labs designed to teach bash scripting concepts, from basic syntax to advanced techniques.

## Contents

- **Basics**: Variables, strings, and operators
- **Control Flow**: Conditionals and loops
- **Functions**: Function definition and usage
- **File Operations**: Reading, writing, and manipulating files
- **Advanced**: Arrays, regular expressions, and error handling

## Getting Started

1. Clone this repository
2. Navigate to a specific lab directory
3. Review the lab description
4. Execute the bash scripts:
    ```bash
    bash script_name.sh
    ```

## Prerequisites

- Bash 4.0 or higher
- Linux/Unix environment or WSL
- Basic command-line knowledge

## How to Use

Each lab includes:
- A problem description
- Starter code (if applicable)
- Expected output examples
